

# Todo-list App

This project is build using react-js with firebase store where you can add ,delete and edit a todo.

********************************************************************************************************************************


 ## Tecnology used : 

 - React js
 - Firbase Store
 - material-ui


### How to Run
 
 - npm install
 - npm start
 
**note** 

You have to add your firbase.js file and add the config from your firebase account
